Projects

'Beyond the Horizon'

As a 2023 participant in the Girls Who Code Summer Immersions program, I
coded a game in Javascript inspired by the theme of progression and hope.
This dramatic fantasy is inspired by the pixelated aspects of Stardew
Valley and old arcade games.

'Cinnamari'

Woo! Cinnamari is an up and coming stationary shop featuring pastel
designs created for children in underpriviledged areas. The intention of this
shop is to create eco-friendly products that can be shipped to local
elementary schools to enable creative learning processes. This mockup was
created in Figma, an effective UI system.

'Mission: Save Keroppi's Birthday!'

Help! Keroppi's birthday party has been affected by a random space invasion!
